Implementation Details

Environment Design

The simulation environment is built using Unity Game Engine, providing a realistic 3D representation of road scenarios. Key components include:

  • Dynamic road networks with varying complexity
  • Realistic physics for vehicle dynamics

Reward Function Design

The reward function is carefully designed to encourage desired driving behaviors:

  • Positive rewards for maintaining lane position and following traffic rules
  • Negative rewards for collisions and off-road driving
  • Reward shaping for smooth acceleration and braking
  • Distance-based rewards to encourage progress toward destinations

PPO Algorithm Implementation

The project utilizes the Proximal Policy Optimization (PPO) algorithm, which offers stable training for continuous control tasks:

  • Clipped objective function to prevent large policy updates
  • Value function loss with entropy bonus for exploration
  • Adaptive learning rate based on KL divergence
  • Generalized Advantage Estimation (GAE) for better value estimation
  • Parallel environment sampling for efficient data collection

Multi-Agent Shared Critic

A shared critic architecture enables efficient learning across multiple agents:

  • Centralized value function that considers the global state
  • Decentralized execution where agents act based on local observations
  • Parameter sharing between agents to reduce learning time

Data Collection & Training

Efficient data collection and training strategies are employed:

  • Prioritized experience replay to focus on challenging scenarios
  • Curriculum learning starting with simple scenarios and gradually increasing complexity
